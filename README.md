# hypertail
Tail a hyperpipe at a hypername

## Usage
```txt
  Usage: hypertail <topic> <key>

  Example:
    $ hypername init startup.world <key>
    $ hypername sync startup.world
    $ hypername list startup.world
    $ hypertail startup.world production.log
```
